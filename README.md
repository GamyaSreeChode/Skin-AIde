# Skin Aide - AI Skin Disease Detection

Skin Aide is an AI-powered skin disease detection system that analyzes skin images and predicts possible skin conditions using deep learning and computer vision techniques.

## Features

- Skin disease prediction using image input
- Deep learning based classification
- User-friendly interface
- Fast and efficient analysis
- AI-integrated healthcare assistance

## Tech Stack

- Python
- Flask
- TensorFlow / Keras
- OpenCV
- HTML/CSS
- JavaScript

```bash
Skin-AIde/

├── static/

```

## Installation

1. Clone the repository

```bash
git clone https://github.com/GamyaSreeChode/Skin-AIde.git
```

2. Navigate into the project folder

```bash
cd Skin-AIde
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

## Usage

- Upload a skin image
- The model analyzes the image
- Predicted skin disease result is displayed
